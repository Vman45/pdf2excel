# pdf2excel
Macros for getting data from PDF tables into excel

Source for this and full documentation can be found at https://exceloffthegrid.com/copy-a-data-table-from-pdf-into-excel/ 

I have simply made a few tweaks. 

ConvertListToTable   Takes a single column (as pasted from a PDF) and splits into columns based on the "NoOfColumns" variable.
